# 基金持仓监控（Fund Portfolio Monitor）

## 触发条件

- 用户要求"基金监控"、"基金快报"、"查看持仓"、"基金净值更新"
- 定时任务（cron）自动执行每日基金监控
- 用户询问特定基金当日表现

## 工作流程

### 步骤 1：确定交易日期

- 确定当前是否为交易日（A股周一至周五，排除法定节假日）
- 最新净值通常为上個交易日收盘后发布（约20:00-22:00更新）
- 若当日为交易日但净值未出，以**上個交易日净值**为准，并在报告中标注
- 同时搜索当日盘中实时行情作为补充

### 步骤 2：批量获取基金净值

**搜索策略**：
- 优先使用 `web_search`，搜索格式：`"{基金全称} {基金代码} 净值 {日期}"`
- 备用格式：`"{基金简称} 最新净值 {日期}"`、`"{基金代码} 净值 日涨跌"`

**数据来源优先级**（web_search 的 description 字段通常已包含关键数据）：
1. 天天基金 / 东方财富（fundf10.eastmoney.com）— 数据最全
2. 新浪财经（stock.finance.sina.com.cn）— 估值与涨跌幅
3. 雪球基金/蛋卷（danjuanfunds.com）— 净值+历史表现
4. 基金公司官网 — 最新净值权威来源
5. 平安证券/同花顺等第三方

**⚠️ 重要限制**：
- `web_extract` 对东方财富、天天基金、多数基金公司官网会返回 `Blocked: URL targets a private or internal network address`
- **不要在 web_extract 上浪费时间**：直接用 web_search 的 description 提取数据
- 若 web_search description 数据不完整，再尝试用 web_extract 访问 danjuanfunds.com 或新浪财经

**🖥️ 浏览器降级方案**（当 web_search + web_extract 都拿不到精确净值时）：
- 用 `browser_navigate` 访问基金公司官方页面（**已验证可用**）：
  - **华夏基金**：`https://www.chinaamc.com/fund/{基金代码}/index.shtml` — NAV 表格直接渲染在无障碍树中，snapshot 即可看到完整历史净值（日期/净值/累计净值）
  - 其他基金公司官网类似，优先尝试 `.shtml` 结尾的页面
- 优势：无需点击、无需登录，页面加载后直接 snapshot 就能拿到净值表格
- 注意：每只基金需单独导航一次，速度较慢，作为最后手段使用

**并行搜索**：一次搜索多只基金（每批4-6只），在 tool call 层面并行发起

### 步骤 3：亏损持仓深度分析

对每只亏损基金计算：
- 当前亏损率
- 回本所需涨幅 = 亏损率 / (1 - 亏损率) × 100%
- 近1月/近3月/今年以来趋势

重点关注亏损 > 10% 的持仓。

### 步骤 4：板块走势分析

搜索以下关键信息：
- 有色/贵金属：搜索 `"有色金属板块 今日走势 黄金白银 {日期}"`
- 恒生科技：搜索 `"恒生科技指数 今日 实时行情 {日期}"`
- A股科技/AI/半导体：搜索 `"A股 科技板块 半导体 AI 今日走势 {日期}"`

### 步骤 5：生成策略建议

按方向给出建议：
- 🪙 有色/金银珠宝：结合地缘政治、美元、通胀分析
- 🇭🇰 恒生科技：结合港股资金流向、估值区间分析
- 🤖 AI/科技：结合美股半导体、A股科创50走势分析

## 输出格式

严格遵循以下格式（用户偏好）：

```
📊 展鹏基金快报 | YYYY年M月D日（周X）

> ⏰ 说明：净值数据来源说明

## 🔴 亏损持仓重点关注

表格：基金 | 代码 | 市值 | 亏损 | 亏损率 | 最新净值 | 日涨跌 | 回本需涨

## 🟢 盈利持仓概览

表格：基金 | 代码 | 最新净值 | 日涨跌 | 近1年 | 今年以来
```

## 关键 Pitfalls

1. **不要用 web_extract 访问东方财富/天天基金/基金公司官网**：100% 被封，改用 web_search 的 description 或 browser_navigate
2. **净值日期要核对**：不同数据源可能相差1-2个交易日，以基金公司官网为准
3. **港股QDII净值滞后**：恒生科技类QDII净值通常T+1或T+2才更新
4. **周末/节假日不更新**：周五净值到周一才有新数据，报告中标注
5. **基金代码要精确**：A类/C类代码不同，净值也不同。常见混淆：永赢科技智选A=022364/C=022365（不是008919/008920），嘉实半导体增强C=014855，科创板芯片ETF联接C（易方达）=020671（不是017469嘉实版），华夏有色金属ETF联接C=016708（不是016707的A类），华夏消费电子ETF联接C=018301
6. **cron 模式下 execute_code 被禁用**：所有操作必须用独立 tool call 完成，并行发起多个 web_search 来加速批量数据获取
